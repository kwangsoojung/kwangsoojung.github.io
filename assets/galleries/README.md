Project galleries are defined in `assets/gallery-manifest.json`.

Use this structure:

- `assets/galleries/shopify/cover.jpg`
- `assets/galleries/shopify/01.jpg`
- `assets/galleries/shopify/02.jpg`
- `assets/galleries/shopify/03.jpg`

- `assets/galleries/brand-campaign/cover.jpg`
- `assets/galleries/brand-campaign/01.jpg`
- `assets/galleries/brand-campaign/02.jpg`

- `assets/galleries/catalog/cover.jpg`
- `assets/galleries/catalog/01.jpg`
- `assets/galleries/catalog/02.jpg`

- `assets/galleries/retail/cover.jpg`
- `assets/galleries/retail/01.jpg`
- `assets/galleries/retail/02.jpg`
- `assets/galleries/retail/03.jpg`

Add more files if needed, then update the matching `images` array in `assets/gallery-manifest.json`.
